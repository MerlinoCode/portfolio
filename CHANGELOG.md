## [1.0.0] - 2023-08-01

### Added
- Initial setup and release of the online portfolio.
- React-based UI architecture with Vite.
- Implementation of React Router for seamless single-page navigation.
- SASS embedded structure for component styling and responsive layouts.
- Interactive particle animations utilizing `tsparticles-slim` and `react-particles`.
- Dynamic project showcasing system with live demo links and tool icons.
- Contact form structure with interactive elements.
- Initial color palette implementation.
