## 📖 About the Project

This is an interactive online portfolio designed to present projects, skills, and contact information dynamically. Carefully crafted with a clean aesthetic and responsive layout, it serves not only as a digital resume but as a showcase of modern frontend development capabilities. The application features internationalization (multi-language support), interactive particle backgrounds, and a component-based architecture for displaying individual projects effectively.

## ✨ Features

- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile devices.
- **Interactive Animations**: Utilizes particles and smooth CSS transitions for a premium user experience.
- **Multi-language Support**: Built-in translations feature to reach audiences worldwide.
- **Project Showcase**: Detailed cards highlighting different projects, their descriptions, and the tools used.
- **Modern Styling**: A customized aesthetic with comprehensive color schemes and Glassmorphism effects.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have Node.js and npm installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/MerlinoCode/online-portfolio-2023.git
   ```
2. Navigate into the project directory
   ```sh
   cd online-portfolio-2023
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
